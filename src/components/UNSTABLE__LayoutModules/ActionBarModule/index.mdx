## Overview

The action bar module can be built using a combination of the `ActionBarModule` and `ActionBarModuleItems` components.

`ActionBarModuleItems` is responsible for positioning the actions, and everything passed in as a child of `ActionBarModuleItems` will be rendered.

## Grid placement

Action bar modules can span between 4 and 16 columns, following the [narrow grid](#references) implementation.

<Row>
  <Column>
    <img alt="#" src="#" />
  </Column>

  <Column>
    <img alt="#" src="#" />
  </Column>
</Row>

```jsx
<Column lg={4}>
  <ActionBarModule>
    <Button kind="ghost">Action 1</Button>

    <ActionBarModuleItems>
      <IconButtonBar
        actions={[
          {
            label: 'Action 2',
            renderIcon: Add16,
          },
          {
            label: 'Action 3',
            renderIcon: Edit16,
          },
          {
            label: 'Action 4',
            renderIcon: Filter16,
          },
        ]}
        size="md"
      />
    </ActionBarModuleItems>
  </ActionBarModule>
</Column>
```
