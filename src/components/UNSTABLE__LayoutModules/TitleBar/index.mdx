# TitleBar

## Overview

Title bars provide...

You can build a title bar using a combination of the `TitleBar`, [`Title`](#references) and `TitleBarItems` components.

`TitleBarItems` is responsible for positioning the actions, and everything you pass in as a child of `TitleBarItems` will be rendered.

## Grid placement

When placing a title bar on the [grid](#references), it should be placed in a minimum of 4 columns and a maximum of 8.

```jsx
<Column lg={4}>
  <TitleBar>
    <Title>Section title</Title>

    <TitleBarItems>
      <IconButtonBar
        actions={[
          {
            label: 'Action 1',
            renderIcon: Add16,
          },
          {
            label: 'Action 2',
            renderIcon: Edit16,
          },
          {
            label: 'Action 3',
            renderIcon: Filter16,
          },
        ]}
        size="md"
      />
    </TitleBarItems>
  </TitleBar>
</Column>
```
