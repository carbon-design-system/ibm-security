# ActionBar

## Overview

Action bars allow users to take multiple actions on the subsequent section.

You can build an action bar using a combination of the `ActionBar` and `ActionBarItems` components.

`ActionBarItems` is responsible for positioning the actions, and everything you pass in as a child of `ActionBarItems` will be rendered.

## Grid placement

When placing an action bar on the [grid](#references), it should be placed in a minimum of 4 columns and a maximum of 16.

```jsx
<Column lg={4}>
  <ActionBar>
    <Button kind="ghost">Action 1</Button>

    <ActionBarItems>
      <IconButtonBar
        actions={[
          {
            label: 'Action 2',
            renderIcon: Add16,
          },
          {
            label: 'Action 3',
            renderIcon: Edit16,
          },
          {
            label: 'Action 4',
            renderIcon: Filter16,
          },
        ]}
        size="md"
      />
    </ActionBarItems>
  </ActionBar>
</Column>
```
