### Overview

The page tab module can be built using a combination of the `PageTabModule`, [`Tabs`](#references), and `PageTabModuleDetails` components.

The `PageTabModule` accepts `Tabs`, and optionally `PageTabModuleDetails`, components as `children` and each `Tab` is responsible for displaying the tab's label and panel content.

### Grid placement

Following the [narrow grid](https://react.carbondesignsystem.com/?path=/docs/grid--narrow) implementation, page tab modules should be placed directly into the grid context.

<Row>
  <Column>
    <img alt="`PageTabModule` do" src={doImage} />
  </Column>

  <Column>
    <img alt="`PageTabModule` don't" src={dont} />
  </Column>
</Row>

```jsx
<Grid>
  <PageTabModule>
    <Tabs>
      <Tab label="Tab 1">
        <Row>
          <Column>Tab 1 content</Column>
        </Row>
      </Tab>
    </Tabs>
  </PageTabModule>
</Grid>
```
