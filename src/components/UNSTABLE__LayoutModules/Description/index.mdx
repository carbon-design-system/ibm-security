# Description

## Overview

Descriptions provide a means to orderly layout short-form content.

You can build a description using a combination of the [`Title`](#references) and `DescriptionContent` components. Everything you pass in as a child of `DescriptionContent` will be rendered.

## Grid placement

When placing a description on the [grid](#references), it should be placed in a maximum of 8 columns.

```jsx
<Column lg={8}>
  <Description>
    <Title>Section title</Title>

    <DescriptionContent>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer finibus
      tortor eget est ornare, a ultrices risus tincidunt. Suspendisse
      condimentum mauris at ornare tempor. Phasellus a arcu ante. Morbi vitae
      ultrices quam, eget eleifend magna. Morbi quis porttitor ex, in elementum
      tellus. In eget rutrum eros, eu scelerisque risus. Phasellus vel pretium
      lorem, ut laoreet sapien. Cras ac purus vitae velit efficitur iaculis.
      Nunc.
    </DescriptionContent>

    <DescriptionContent>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer finibus
      tortor eget est ornare, a ultrices risus tincidunt. Suspendisse
```
