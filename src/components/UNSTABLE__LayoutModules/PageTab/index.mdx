# PageTab

## Overview

Page tab provides orderly tab navigation for a page.

You can build a page tab using a combination of the `PageTab`, [`Tabs`](#references), and `PageTabDetails` components.

The `PageTab` accepts `Tabs`, and optionally `PageTabDetails`, components as `children` and each `Tab` is responsible for displaying the tab's label and panel content.

## Grid placement

When placing a page tab on the [grid](#references), it should be placed directly in to the grid context.

```jsx
<Grid>
  <PageTab>
    <Tabs>
      <Tab label="Tab 1">
        <Row>
          <Column>Tab 1 content</Column>
        </Row>
      </Tab>

      <Tab label="Tab 2" />
      <Tab label="Tab 3" />
    </Tabs>
  </PageTab>
</Grid>
```
