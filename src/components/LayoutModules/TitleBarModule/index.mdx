### Component API

<ArgsTable />

_**\***Required property_

Additional props passed into `TitleBarModule` will be forwarded along to the underlying title bar module containers.

#### `subsection`

Use the `subsection` prop to specify whether a subsection should be used.

<Canvas withSource="none">
  <Default title="Sub-section title" subsection parameters={parameters} />
</Canvas>

```jsx
<TitleBarModule title="Sub-section title" subsection />
```

#### `element`

This prop specifies a different element to be rendered as a title.

<Canvas withSource="none">
  <Default element="h3" title="Section 3 title" parameters={parameters} />
</Canvas>

```jsx
<TitleBarModule element="h3" title="Section 3 title" />
```

### Anatomy

1. **Title:** This the title itself, learn more about potential [variants](#variants) above.
2. **Top spacer:** There is 16px spacer `$spacing-05`, included at the top of the TitleBarModule.
3. **Bottom spacer:** The 16px spacer `$spacing-05`, included inside the bar, means that the module can be placed flush to edge of the components below.

<img alt="`TitleBarModule` anatomy" src={anatomy} />

### Variants

Actions can be placed inline with the title and a label can also be included below.

The title bar can accept two type variants, sub-section title and section title, so you shouldn't supplement your own type style.

### Grid placement

Title bar modules span a minimum of 4 columns, following the [narrow grid](https://react.carbondesignsystem.com/?path=/docs/grid--narrow) implementation.

  <Column>
    <img alt="`TitleBarModule` donʼt" src={dont} />
  </Column>
</Row>

```jsx
<Column lg={4}>
  <TitleBarModule title="Section title" />
</Column>
```
