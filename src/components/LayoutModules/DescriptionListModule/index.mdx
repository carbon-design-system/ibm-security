### Anatomy

1. **Term:** This is the label, it should left aligned to a grid column.
2. **Gutter:** There is always a 32px gutter between the ‘term’ and ‘description’ columns. In this way, the ‘description’ column is always offset from the term column by exactly 32px.
3. **Description:** The description placement is relative to the term label, it will always fall 32px to the right of the term.
4. **Bottom spacer:** The 16-pixel spacer `$spacing-05` below the ‘Row spacer’ row provides the starting point for the layout module positioned directly underneath it. At the end of the description list module both the ‘Row spacer’ and ‘Bottom spacer’ join together to create a 32px buffer.

### Grid placement

Description list modules span a minimum of 4 columns, following the [narrow grid](https://react.carbondesignsystem.com/?path=/docs/grid--narrow) implementation. The ‘term’ column should left align with the grid, the ‘description’ column will be offset from the ‘term’ column by exactly 32px. It may or may not align to the grid, don’t force it.

### Layout considerations

Occassionally [spacing tokens](https://carbondesignsystem.com/guidelines/spacing/overview#spacing-scale) will need to be included to offset the description list module to ensure it top-aligns with the adjacent layout container. If top spacing tokens are necessary, reference the layout module that sits directly above it as the starting point for placement.
