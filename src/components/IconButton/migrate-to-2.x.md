# `IconButton`

This now uses [`forwardRef`](https://reactjs.org/docs/forwarding-refs.html) to manage focus and may affect applications that depend on the old behavior.

## Props

| `1.x`      | `2.x`                                                  |
| ---------- | ------------------------------------------------------ |
| `iconName` | `renderIcon` (see [`Icon`](../Icon/migrate-to-2.x.md)) |
